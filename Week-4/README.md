# Week 4 - HTML Basics

## Objective

Create basic static webpages using HTML tags, links, lists, headings, paragraphs, and images.


## Assignment Details

## Concepts Practiced

- HTML boilerplate structure
- Metadata using `meta` tags
- Page title using `title`
- Headings and text hierarchy
- Creating hyperlinks
- Displaying images
- Ordered and unordered lists
- Basic webpage content organization


## Outcome

This week builds the foundation for creating simple static webpages using common HTML elements.
