# Week 6 - React Forms and MERN Employee App

## Objective

Build React applications using forms, validation, routing, state management, and connect a React frontend to an Express and MongoDB backend.


## Work Completed

### Product Form App

Location: `Product`

- `src/App.jsx`
- `src/components/FormDemo.jsx`

- Created a React app using Vite.
- Used component-based structure.
- Created a user form.
- Used `react-hook-form` for form handling.
