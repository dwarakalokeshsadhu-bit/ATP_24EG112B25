# Week 6 - React Forms and MERN Employee App

## Objective

Build React applications using forms, validation, routing, state management, and connect a React frontend to an Express and MongoDB backend.


## Work Completed

- Created a React app using Vite.
- Used component-based structure.
- Created a user form.
- Used `react-hook-form` for form handling.


Backend features:

- Express server setup.
- MongoDB connection using Mongoose.
- Environment variables using `dotenv`.

Frontend features:

- React Router based navigation.
- Nested routes under `RootLayout`.
- Employee creation page.


## Outcome

This week strengthens React form handling and introduces a complete MERN CRUD workflow with separate frontend and backend applications.
